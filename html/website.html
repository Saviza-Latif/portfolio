Radiant Events

overview

This project is comprehensive event organizer website designed to streamline the operations of a event organizer.

languages/tools

HTML , CSS , JavaScript , PHP , mysql , XAMPP , VScode

keyfeatures

  • Secure login and signup functionality
  • User can leave review or send message to admin
  • Comprehensive information on available event packages
  • Admin can add and track order
  • Admin can view and delete user, review, messages

Future improvement

  • Adding new online ordering services
  • Enhancing the gallery with videos and highlights of pervious events
